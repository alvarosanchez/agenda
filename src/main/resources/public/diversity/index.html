Diversity Tickets

This diversity program is aimed to help underrepresented groups in tech to attend and lowering the barriers for them. This includes but isn’t limited to: women, people of colour, LGBTQIA+ people, and disabled people. Your privacy, details and answers will only be shared with the organisers of Greach 2020. We will not share any data with those who sponsor these tickets. This application is for one conference ticket only. If you want two tickets because you prefer to come with a friend to the conference please add that comment in the form below. Tickets are valid for the 3 days of the conference

All attendees, speakers, sponsors and staff at Greach 2020 are required to agree with the code of conduct of the event. Organizers will enforce this code throughout the event.

Application closes January 13th 2020.

Winner will be contacted January 20th 2020.
